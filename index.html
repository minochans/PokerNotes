プライバシーポリシー

```

本プライバシーポリシーは、PokerNotes（以下「本アプリ」）におけるユーザー情報の取り扱いについて説明するものです。

1. 収集する情報

本アプリは、ポーカーのハンド履歴、プレイヤー名、スタック、アクション、メモ、タグ、設定情報などを、ユーザーの端末内に保存します。 これらのユーザー入力データは、開発者のサーバーへ送信されません。

本アプリでは、広告表示および広告に関する同意管理のため、Google AdMob、Google Mobile Ads SDK、Google User Messaging Platform SDKを使用しています。 これらのSDKにより、広告識別子、端末情報、大まかな位置情報、広告の表示・操作に関する情報、クラッシュ情報、パフォーマンス情報などが収集される場合があります。

2. 利用目的

本アプリで扱う情報は、以下の目的で利用されます。

  • 本アプリの機能提供
  • ユーザーが入力した記録の保存・表示
  • 広告の表示
  • 広告配信の最適化
  • アプリの品質改善、不具合調査、パフォーマンス改善
  • 法令、規約、各プラットフォームのポリシーへの対応

3. 第三者サービスの利用

本アプリでは、以下の第三者サービスを利用しています。

  • Google AdMob
  • Google Mobile Ads SDK
  • Google User Messaging Platform SDK
  • Apple App Store / StoreKit

これらの第三者サービスが収集する情報、利用目的、管理方法については、各サービス提供者のプライバシーポリシーおよび関連文書をご確認ください。

  • Google プライバシーポリシー
  • Google AdMob に関する情報
  • Apple プライバシーポリシー

4. トラッキングについて

本アプリでは、広告の表示および広告効果測定のため、ユーザーの許可を得たうえで、他社のアプリやWebサイトを横断したトラッキングに利用される情報を取得する場合があります。

ユーザーは、iOSの設定アプリから、いつでもトラッキング許可の状態を変更できます。

5. データの保存

ユーザーが本アプリ内で作成した記録データは、原則としてユーザーの端末内に保存されます。 アプリを削除した場合、端末内に保存されたデータも削除される場合があります。

6. ユーザーの選択

広告に関する同意が必要な地域では、本アプリ内で表示される同意画面またはプライバシー設定画面から、広告に関する選択を変更できる場合があります。

また、iOSの設定アプリから、アプリごとのトラッキング許可を変更できます。

7. 未成年の利用について

本アプリは、13歳未満の児童を対象として意図的に個人情報を収集するものではありません。
```
